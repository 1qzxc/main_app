Kubernetes

Repository

Using examples on the github I have set up my easily scallable, constistent kubernetes cluster that could be brung up in the matters of minutes using Vagrant, Virtualbox Ansible

Cluster is provisioned using Ansible. The playbook installs kubernetes using kubeadm and also applies Calico CNI config so the network works as soon as cluster is up.
