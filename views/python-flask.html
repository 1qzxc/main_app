Flask blogging app

Check it out


This is a study project on python app development tools and practices using flask and flaskr-tdd tutorial. I worked through the tutorial step by step and have working flask app.

In this app you are supposed to log in using hardcoded credentials, since there is no database associated with it. You would log in using "admin - admin" login password pair, after that you may leave posts.

Whenever you add a new entry, it is shown under the form, along with other entries. If you log out, all the entries would be shown on the main page.

If you log out, all the entries would be shown on the main page.

The entries are kept in a SQLite database.
